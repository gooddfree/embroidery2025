Image to Embroidery Pattern Converter

This project is a web-based tool that allows users to convert images into simplified embroidery patterns. The tool processes uploaded images and converts them into a grid-based black-and-white design, ideal for embroidery use.

Features

Image Upload: Upload images directly from your device.

Grid Conversion: Converts images into a grid-based black-and-white pattern.

Preview: View the generated embroidery pattern instantly.

Technologies Used

HTML: Structuring the webpage.

CSS: Styling the interface.

JavaScript: Processing images and generating patterns.

How to Use

Open the website in your browser.

Click on the Upload Your Image section and select an image file (e.g., .jpg, .png).

Click the Convert to Pattern button.

View the generated embroidery pattern displayed on the screen.

Installation

Download the index.html file from this repository.

Open the file in any modern web browser.
